# Contributing

Thanks for your interest in contributing!

## How to contribute
- Open an issue for bugs, questions, or feature ideas.
- For changes, open a pull request (PR).

## Pull request guidelines
- Keep PRs focused and small when possible.
- Describe the problem and the solution.
- Include steps to reproduce and verify.
- Add screenshots or GIFs for UI changes if applicable.

## Code style
- Follow existing project structure and naming.
- Keep scripts minimal and readable.
